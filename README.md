# Project Title

Tomato Diseases

## Getting Started

### Pre-requisites
* Python 2.7
* Tensorflow
* Tensorboard
* Numpy
* Matplotlib



### Installing

#### Pay attention: Needs the dataset to retrain.

Initialize Tensorboard

```
tensorboard --logdir tf_files/training_summaries &
```

Run the train.sh to retrain the network

```
bash train.sh
```

If want to edit parameters with vi/nano/gedit 
```
vi train.sh
```
